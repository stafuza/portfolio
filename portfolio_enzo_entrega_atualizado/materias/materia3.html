Técnicas Avançadas de Programação Web e Mobile

Professor: Marcelo Tadeu Boer (Especialista)

Sobre a Matéria

A disciplina de Técnicas Avançadas de Programação aprofunda os conhecimentos em desenvolvimento de software, explorando conceitos e práticas que vão além da programação básica. O foco está em melhorar a qualidade, eficiência e manutenibilidade do código.

Nesta matéria, são estudados tópicos como programação orientada a objetos avançada, padrões de projeto (Design Patterns), princípios SOLID, programação funcional, programação reativa, e técnicas de refatoração. Os alunos aprendem a identificar e aplicar o padrão mais adequado para cada problema, resultando em código mais limpo, modular e extensível.

A disciplina também aborda aspectos importantes como testes automatizados, integração contínua, análise estática de código, e técnicas para lidar com código legado. Os estudantes desenvolvem projetos práticos que aplicam esses conceitos em cenários realistas, preparando-os para enfrentar desafios complexos de desenvolvimento de software.

React

Nesta disciplina, exploramos frameworks modernos para desenvolvimento web e mobile, com foco em React para criação de interfaces dinâmicas e integração com back-end via Firebase.
