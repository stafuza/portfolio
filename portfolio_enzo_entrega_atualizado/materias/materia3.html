Sobre a Matéria

A disciplina de Técnicas Avançadas de Programação Web e Mobile explora os conceitos e tecnologias mais recentes para o desenvolvimento de aplicações modernas para web e dispositivos móveis. O foco está em proporcionar aos alunos conhecimentos avançados que vão além do desenvolvimento básico.

Nesta matéria, são abordados frameworks e bibliotecas populares como React, Angular, Vue.js para desenvolvimento web, e React Native, Flutter e Kotlin/Swift para desenvolvimento mobile. Os alunos aprendem a criar interfaces responsivas, implementar APIs RESTful, gerenciar estados de aplicação, e integrar serviços de backend.

Além disso, a disciplina cobre tópicos como Progressive Web Apps (PWAs), Single Page Applications (SPAs), otimização de performance, segurança, e boas práticas de desenvolvimento. Os estudantes desenvolvem projetos práticos que simulam cenários reais do mercado de trabalho.
