Dispositivos de Entrada/Saída e Barramentos

Matéria: Organização de Computadores e Sistemas Operacionais • Data: 15/05/2025

Dispositivos de Entrada/Saída e Barramentos

Neste módulo, exploramos os periféricos utilizados para entrada e saída de dados em sistemas computacionais e os barramentos responsáveis pela comunicação entre os componentes internos e externos do computador.

Dispositivos Externos

Dispositivos externos, ou periféricos, são equipamentos que permitem a interação com o computador, seja inserindo, recebendo ou armazenando dados.

  • Entrada: Teclados, mouses, scanners, webcams, microfones;
  • Saída: Monitores, impressoras, alto-falantes;
  • Armazenamento: HDs externos, pen drives, DVDs;
  • Comunicação: Modems, roteadores.

Conexões e Comunicação

Os periféricos se conectam ao sistema por módulos de E/S e interfaces como USB, HDMI, Ethernet ou via conexões sem fio como Wi-Fi e Bluetooth.

  • Após conectar, o sistema operacional instala os drivers para controlar o dispositivo;
  • A transferência de dados é feita por módulos que garantem comunicação segura e eficiente.

Gerenciamento e Desafios

O gerenciamento dos dispositivos envolve controladores de E/S (drivers) e o monitoramento de seu desempenho pelo sistema operacional.

  • Compatibilidade: Dispositivos precisam ser reconhecidos pelo SO;
  • Desempenho: O sistema deve evitar atrasos e conflitos;
  • Segurança: Dispositivos externos podem representar riscos se não forem confiáveis.

Barramentos

Barramentos são caminhos físicos ou lógicos que transportam dados entre os componentes do sistema.

  • Barramento de Dados: Transfere os dados propriamente ditos (8 a 128 bits por ciclo);
  • Barramento de Endereços: Define os locais de leitura/escrita;
  • Barramento de Controle: Envia sinais para coordenar as transferências.

Tipos de Barramentos

Os barramentos são classificados por sua localização e aplicação no sistema.

  • Barramento do Sistema: Conecta a CPU à memória e aos controladores de E/S;
  • Barramento de Expansão: Conecta placas como GPU, som ou rede (PCI, PCIe);
  • Barramento Local: Ex: SATA para comunicação com HDs/SSDs.

Exemplos de Barramentos

São exemplos comuns de tecnologias de barramento:

  • PCI / PCI Express (PCIe): Comunicação de alta velocidade para placas internas;
  • USB: Padrão universal para conectar periféricos externos;
  • SATA: Conexão de dispositivos de armazenamento internos (HDDs e SSDs).

Funcionamento e Sincronismo

A comunicação nos barramentos pode ser:

  • Sincronizada: Com base em um clock do sistema;
  • Assincronizada: Dispositivos controlam o fluxo por sinais próprios.
