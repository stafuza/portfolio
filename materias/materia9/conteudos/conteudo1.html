Organização de Computadores e Sistemas Operacionais

Nesta disciplina, exploramos os componentes internos de um computador e o funcionamento dos sistemas operacionais. Abordamos memórias, registradores, CPU, hierarquia de armazenamento e gerenciamento de memória virtual.

Componentes Fundamentais de um Computador

Estudamos os principais elementos de hardware de um sistema computacional:

  • CPU (Unidade Central de Processamento): executa instruções e controla as operações do sistema.
  • Memória RAM e ROM: armazenam dados temporários e instruções permanentes.
  • Dispositivos de Entrada/Saída: como teclado, mouse, monitor e impressora.
  • Barramentos: canais de comunicação entre os componentes.

Ciclo de Processamento da CPU

A CPU executa o ciclo básico de instrução em quatro etapas:

  • Fetch: busca a instrução na memória.
  • Decode: decodifica a instrução.
  • Execute: realiza a operação.
  • Store: armazena o resultado.

Hierarquia de Memória

A hierarquia de memória organiza diferentes tipos de armazenamento de acordo com a velocidade e o custo:

  • Registradores (mais rápidos e internos à CPU);
  • Cache (níveis L1, L2, L3);
  • RAM (memória principal);
  • Armazenamento em Disco (HD/SSD).

Memória Cache

A memória cache armazena dados frequentemente acessados pela CPU, reduzindo o tempo de acesso à RAM.

  • Cache Hit: quando os dados estão na cache.
  • Cache Miss: quando os dados precisam ser buscados na RAM.
  • Algoritmos de substituição: FIFO, LRU, LFU, Random.

Registradores

Os registradores são unidades de armazenamento extremamente rápidas dentro da CPU, usados para armazenar dados e instruções temporárias.

  • Registradores de dados: armazenam operandos.
  • Registradores de endereço: apontam para locais na memória.
  • Registradores de controle: armazenam estados da CPU.
"Registradores participam de todas as etapas do ciclo de instrução: busca, decodificação, execução e armazenamento."

Memória Virtual

A memória virtual permite que sistemas usem espaço no disco como se fosse RAM, aumentando a capacidade de execução de programas.

  • Utiliza técnicas como paginação e mapeamento de páginas.
  • Quando a página não está na RAM, ocorre um Page Fault.
  • Usa algoritmos como FIFO, LRU e MRU para substituir páginas.

Sistemas Operacionais

Os sistemas operacionais gerenciam hardware e fornecem uma interface para o usuário. Suas funções incluem:

  • Gerenciamento de processos e memória;
  • Controle de dispositivos de entrada/saída;
  • Agendamento de tarefas e comunicação entre processos.
